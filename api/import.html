    # Import

    Importing your text to your dataset is done via API.

    # Endpoint

    POST /api/import

    # Request body

    NameTypeDescription
    datasetstringThe dataset which the text should be uploaded.
    ID can be found on the dataset detail page
    textstringThe text later used for annotation.
    {
  "dataset": "5693561773228032",
  "text": "The Vostok programme was a Soviet human spaceflight project."
}

    # Response

    If successful, the response statuscode is 200
